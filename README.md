# TokTalk

스마트인재개발원에서 주관하는 실전프로젝트에서 사용한 서버코드입니다.
Spring Boot(gradle)를 활용하여 외국어 챗봇 서비스의 대시보드를 구현하였습니다.

## 📟 프로젝트 소개
"LLM와 번역 API를 이용한 공공기관 외국어 챗봇 서비스"를 주제로 한 프로젝트입니다. 외국인들의 민원처리를 수월하게 하기 위한 챗봇을 제작해보았습니다.
